# Task-CRUD-Passport-
A task manager site, which allows registered users to create, read, update and delete tasks. Registration and Authentication is done using passport.js

# Tech Stack
1. Nodejs
2. Express
3. Flash
4. Passportjs
5. Mongoose
6. Mongoatlas
7. Heroku
8. Git
9. HTML, CSS
